# 剧情大纲（第1-10章）

## 📊 总体结构

```
第1-2章：起源篇 — 缅北逃生 + 系统激活
第3-5章：都市篇 — 回归复仇 + 建立势力
第6-7章：升级篇 — 天道楼 + 恐惧完成
第8-10章：跨界篇 — 武侠世界 + 回归备战
```

---

## 📖 各章概要

### 第1章：缅北地狱

- [[陈天]]被骗到缅北诈骗园区3个月
- [[波觉]]威胁50万赎身
- 凌晨3点逃出，杀波觉，取证据
- 濒死之际激活[[大反派系统]]
- [[灭世魔尊·虚无]]登场

**核心情绪**：绝望 → 求生 → 觉醒

---

### 第2章：系统觉醒

- 系统正式激活，领取新手礼包
- 回到江南市
- 救下[[林诗雨]]
- 提示"真正的猎场在国门之内"

**核心情绪**：迷茫 → 确认方向

---

### 第3章：招聘会

- 遇到三个叛友：[[王浩]][[李强]][[张伟]]
- 用[[恐惧凝视]]让他们在众人面前崩溃
- [[萧辰]]首次警告

**核心情绪**：复仇快感 + 树敌

---

### 第4章：苏家招亲（上）

- 苏家招亲大会
- [[陈天]]参加
- 文试/武试开始

**核心情绪**：野心 + 对抗

---

### 第5章：苏家招亲（下）+ 暗夜初成

- [[陈天]]击败[[萧辰]]夺魁
- [[苏清雪]]提出追求条件
- 正式建立[[暗夜]]势力
- [[影刃]][[夜鸦]]加入

**核心情绪**：胜利 + 布局

---

### 第6章：天道楼

- [[天道楼]]观察者出现
- 神秘组织开始监视[[陈天]]
- 陈天意识到更大的威胁

**核心情绪**：危机感 + 深层世界观

---

### 第7章：恐惧完成

- 对[[萧辰]]使用[[恐惧凝视]]
- 萧辰精神彻底崩溃
- 获得特殊奖励：[[恐惧领域]]
- 萧辰被送往疗养院

**核心情绪**：碾压 + 能力升级

---

### 第8章：跨界

- [[跨界通行证]]冷却完成
- 穿越到武侠世界（南宋临安）
- 能力被压制到三流武者
- 精神类/身法类不受压制

**核心情绪**：新世界 + 探索

---

### 第9章：江湖风暴

- 挑战武侠世界群雄
- 击败丐帮高手
- [[方云鹏]]亲自出场
- [[恐惧领域]]震撼全场

**核心情绪**：横扫 + 威慑

---

### 第10章：归来

- 回到现实世界
- [[萧辰]]精神崩溃，被断绝关系
- [[萧战天]]秘密前往京城
- [[苏清雪]]成为代理董事长
- [[暗夜]]继续壮大
- 陈天准备迎接更大挑战

**核心情绪**：蓄势 + 新篇章

---

## 🔄 主线节奏

```
生存危机 → 系统觉醒 → 复仇 → 扩张 → 升级 → 跨界 → 回归 → 备战
     ↑                                                              ↓
     └─────────────── 螺旋上升（每个循环更强）←──────────────────────┘
```
