## 🔄 跨界规则

| 规则 | 说明 |
|------|------|
| **能力压制** | 跨界后能力压制到当地水平（三流武者） |
| **精神类例外** | [[恐惧凝视]]等精神功法不受压制 |
| **身法例外** | [[魅影步]]等身法不受压制 |
| **时间流速** | 跨界5天 = 现实约5天 |
| **冷却期** | 30天后才能再次跨界 |
