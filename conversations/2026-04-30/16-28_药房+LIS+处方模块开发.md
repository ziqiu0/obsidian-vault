**日期:** 2026-04-30
**用户:** 湫

## 关键内容

### 1. 药房管理系统（完整）
- **数据库**: drug_catalog / drug_stock_batch / drug_stock_movement / drug_purchase_order / drug_purchase_item / dispensing_record 共6张表
- **完整流程**: 新增药品→采购入库→审核→入库(自动创建批次+库存变动)→发药(FIFO选批次扣库存)
- **前端**: 药品目录页(DrugCatalog.vue) + 采购入库页(PurchaseOrder.vue)
- **后端**: 30个文件（6实体+6仓储+12服务+6控制器）

### 2. LIS 检验系统
- **数据库**: lis_test_item / lis_test_request / lis_test_request_item / lis_sample / lis_test_result / lis_report 共6张表
- **流程**: 医生开检验申请→工作列表→样本采集→核收→结果录入→报告审核→报告发布
- **前端API**: 已封装全部接口，前端的 4 个 LIS 页面可以直接对接
- **后端**: 17个文件（6实体+6仓储+5控制器）

### 3. 处方模块
- **数据库**: prescription / prescription_item 共2张表
- **流程**: 医生选患者→开处方(选药品+用量+频次+天数)→保存草稿
- **状态**: DRAFT → 待缴费(PAID) → 发药(DISPENSED)
- **后端**: 5个文件（2实体+2仓储+1控制器）

### 4. 医生诊间工作站更新
- MyPatients.vue 已添加**开处方**和**开检验**按钮（VISITED状态显示）
- 开处方弹窗：搜索药品→填写用量/频次/天数→保存
- 开检验弹窗：搜索检验项目→提交→自动创建样本记录
- 新增 API 模块：`api/doctor.js`（处方+检验申请+报告查询）
