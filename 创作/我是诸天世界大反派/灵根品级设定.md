# 《我是诸天世界大反派》灵根品级设定
[[novels/chapters/第三章_青云夺缘|第三章_青云夺缘]]
在青云修真界，灵根品级从低到高分为九等：

| 品级     | 灵根类型 | 资质说明        | 修行上限                       |
| ------ | ---- | ----------- | -------------------------- |
| **九流** | 五灵根  | 资质驳杂        | 终其一生也难窥金丹门径，大多只能做个外门弟子了此残生 |
| **八流** | 四灵根  | 比五灵根稍好      | 若无非凡机缘，最多也就止步筑基            |
| **七流** | 三灵根  | 资质中等        | 青云宗外门的主要构成                 |
| **六流** | 双灵根  | 天赋异禀        | 足以成为内门核心弟子                 |
| **五流** | 单灵根  | 天才中的天才，百年难遇 | 宗门必定倾尽资源培养                 |
| **四流** | 先天灵根 | 直指大道，千年一出   | 必成渡劫大能                     |
| **三流** | 混沌灵根 | 可塑性极强       | 未来不可限量                     |
| **二流** | 先天道体 | 万法不侵，万中无一   | 长生有望                       |
| **一流** | 无灵根  | 传说中的存在，返璞归真 | 大道根源                       |

## 备注

苏清月的天生道体已是二流顶尖，千年一出，若是栽培得当，未来飞升仙界也不是不可能。
