## 📚 学习内容

### MSA 论文 (Memory Sparse Attention)
- 论文：arXiv:2603.23516
- 核心：稀疏潜状态记忆，支持 100M token 上下文
- 已创建详细笔记：`conversations/MSA-Memory-Sparse-Attention.md`

**核心技术：**
1. Top-k Chunk 选择稀疏注意力
2. Document-wise RoPE（解决训练-推理长度不匹配）
3. 三阶段推理：Encoding → Routing → Generation
4. Memory Interleave（多跳迭代推理）
5. 分层存储：K̄ᴿ 在 GPU，K̄/V̄ 在 CPU DRAM

**实验结果：**
- NIAH @ 1M token：**94.84%**（基线仅 24.69%）
- 超越 RAG +16.0%
- 16K→100M 衰减 < 9%
