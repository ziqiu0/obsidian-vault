# 零售订单接口 (Order)

基础路径: `/api/orders`

## 接口列表

### GET /my - 获取当前用户订单列表

**需要认证:** 是

**查询参数:**
- `page`: 页码
- `limit`: 每页数量
- `status`: 状态筛选（可选）

**响应:**
```json
{
  "code": 200,
  "data": {
    "orders": [...],
    "total": 总数,
    "page": 1,
    "totalPages": 5
  }
}
```

### GET /:id - 获取订单详情

**需要认证:** 是

**响应:**
```json
{
  "code": 200,
  "data": {
    "order": {
      "orderNo": "...",
      "items": [...],
      "totalAmount": 200,
      "status": "...",
      // ...
    }
  }
}
```

### POST /create - 创建订单

**需要认证:** 是

**请求体:**
```json
{
  "items": [
    {
      "productId": "...",
      "quantity": 2
    }
  ],
  "address": {
    "receiver": "...",
    "phone": "...",
    "province": "...",
    "city": "...",
    "district": "...",
    "detail": "..."
  },
  "remark": "..."
}
```

**响应:**
```json
{
  "code": 200,
  "message": "创建成功",
  "data": {
    "order": { /* 订单对象 */ }
  }
}
```

### PUT /:id/cancel - 取消订单

**需要认证:** 是

**响应:** 更新后的订单
