# 商品接口 (Product)

基础路径: `/api/products`

## 接口列表

### GET / - 获取商品列表

**不需要认证**

**查询参数:**
- `page`: 页码，默认 1
- `limit`: 每页数量，默认 10
- `category`: 分类筛选（可选）
- `keyword`: 搜索关键词（可选）

**响应:**
```json
{
  "code": 200,
  "data": {
    "products": [...],
    "total": 总数,
    "page": 1,
    "totalPages": 10
  }
}
```

### GET /:id - 获取商品详情

**不需要认证**

**响应:**
```json
{
  "code": 200,
  "data": {
    "product": {
      "_id": "...",
      "name": "...",
      "description": "...",
      "price": 100,
      "stock": 1000,
      // ... 完整商品信息
    }
  }
}
```

### POST / - 创建商品 (后台管理)

**需要认证:** admin

**请求体:** 完整商品信息

**响应:** 创建成功的商品数据

### PUT /:id - 更新商品

**需要认证:** admin

**请求体:** 更新字段

### DELETE /:id - 删除商品

**需要认证:** admin
