# 认证接口 (Auth)

基础路径: `/api/auth`

## 接口列表

### POST /register - 用户注册

**请求体:**
```json
{
  "username": "string",
  "password": "string",
  "nickname": "string",
  "email": "string (可选)",
  "phone": "string (可选)",
  "role": "string (可选，默认 user)"
}
```

**响应:**
```json
{
  "code": 200,
  "message": "注册成功",
  "data": {
    "token": "jwt_token",
    "user": {
      "_id": "...",
      "username": "...",
      "nickname": "...",
      "role": "...",
      "avatar": "..."
    }
  }
}
```

### POST /login - 用户登录

**请求体:**
```json
{
  "username": "string",
  "password": "string"
}
```

**响应:**
```json
{
  "code": 200,
  "message": "登录成功",
  "data": {
    "token": "jwt_token",
    "user": {
      "_id": "...",
      "username": "...",
      "nickname": "...",
      "role": "...",
      "avatar": "..."
    }
  }
}
```

**错误情况:**
- 用户名不存在 → `code: 400`, 消息: "用户名或密码错误"
- 密码错误 → `code: 400`, 消息: "用户名或密码错误"

### GET /profile - 获取当前用户信息

**需要认证:** 是

**响应:**
```json
{
  "code": 200,
  "data": {
    "user": {
      "_id": "...",
      "username": "...",
      "nickname": "...",
      "role": "...",
      "email": "...",
      "phone": "...",
      "avatar": "..."
    }
  }
}
```

### PUT /profile - 更新用户信息

**需要认证:** 是

**请求体:**
```json
{
  "nickname": "string (可选)",
  "email": "string (可选)",
  "phone": "string (可选)",
  "avatar": "string (可选)"
}
```

## JWT Token 内容

```javascript
{
  "userId": "用户ID",
  "role": "用户角色",
  "iat": 时间戳,
  "exp": 过期时间戳
}
```

默认有效期: 通常 7 天
