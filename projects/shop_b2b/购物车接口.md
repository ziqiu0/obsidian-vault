# 购物车接口 (Cart)

基础路径: `/api/cart`

## 接口列表

### GET / - 获取用户购物车

**需要认证:** 是

**响应:**
```json
{
  "code": 200,
  "data": {
    "cart": {
      "items": [
        {
          "product": { /* 完整商品信息 */ },
          "quantity": 2,
          "selected": true
        }
      ],
      "totalItems": 2,
      "totalPrice": 200
    }
  }
}
```

### POST /add - 添加商品到购物车

**需要认证:** 是

**请求体:**
```json
{
  "productId": "商品ID",
  "quantity": 1
}
```

**响应:** 更新后的购物车

### PUT /update/:itemId - 更新购物车商品数量

**需要认证:** 是

**请求体:**
```json
{
  "quantity": 3
}
```

**响应:** 更新后的购物车

### DELETE /remove/:itemId - 删除商品

**需要认证:** 是

**响应:** 更新后的购物车

### POST /clear - 清空购物车

**需要认证:** 是

**响应:** 空购物车
