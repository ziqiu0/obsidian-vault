# 供货订单接口 (SupplyOrder)

基础路径: `/api/supply-orders`

## 查询接口

### GET / - 获取供货订单列表

**需要认证:** 是

**查询参数:**
- `page`: 页码
- `limit`: 每页数量
- `status`: 状态筛选
- `supplierId`: 供应商筛选 (管理员)

**权限:**
- 供应商: 只能看到自己的订单
- 管理员/采购员: 看到所有订单

### GET /:id - 获取供货订单详情

**需要认证:** 是

**响应:** 完整订单详情，包含需求和报价信息

## 供应商操作

### POST /:id/ship - 发货

**需要认证:** 是 (供应商本人)

订单状态从 `approved` → `shipped`

记录发货时间 `shippedAt`

## 采购方操作

### POST /:id/confirm - 确认收货

**需要认证:** 是 (admin/procurement)

订单状态从 `shipped` → `confirmed` → `completed`

记录确认时间 `confirmedAt` 和完成时间 `completedAt`

### POST /:id/cancel - 取消订单

**需要认证:** 是 (admin/procurement)
