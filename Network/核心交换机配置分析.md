# 核心交换机配置分析

> **设备**: H3C S7503X | **型号**: Core_S7503X
> **版本**: 7.1.070, Release 7755P01
> **来源**: 设备导出配置 `内外网核心交换机.txt`

---

## 1. IRF 堆叠架构（核心要点）

两台 S7503X 通过万兆口实现 **IRF 双机堆叠**：

| 成员 | 优先级 | 堆叠口 | 物理端口 | MAD |
|:----:|:------:|:-------|:---------|:---:|
| Member 1 | 32（主） | irf-port 1/2 | XGE1/2/0/40 | Vlan4000: 192.168.168.1/30 |
| Member 2 | 20（备） | irf-port 2/1 | XGE2/2/0/40 | Vlan4000: 192.168.168.2/30 |

- 堆叠链路：`Ten-GigabitEthernet1/2/0/40 ↔ Ten-GigabitEthernet2/2/0/40`
- MAD BFD 检测：通过 Vlan4000（端口 GE1/2/0/1 + GE2/2/0/1）
- STP 根桥：Member 1 为 `stp instance 0 root primary`

## 5. 安全策略

### 外网 ACL 3999（Vlan11 入方向过滤）

```text
rule 5  permit ip source 192.168.1.134 0        # 财务1
rule 10 permit ip source 192.168.1.135 0        # 财务2
rule 15 permit ip source 192.168.1.153 0        # ? 
rule 20 permit ip source 192.168.1.152 0        # ?
rule 100 deny ip destination 10.35.17.23 0      # 禁止访问此地址
rule 200 permit ip                              # 其他全部放行
```

> 目前外网(Vlan11)只限制了访问 10.35.17.23，其他全部放行。**权限较宽松**。

### 路由

```text
ip route-static 0.0.0.0 0 192.168.200.1           # 默认路由 → 出口路由器
ip route-static 195.4.20.0 24 192.168.200.99       # 特殊路由（公网IP→内网）
ip route-static 197.4.20.0 24 192.168.200.99
ip route-static 200.200.200.0 24 192.168.200.99
```
